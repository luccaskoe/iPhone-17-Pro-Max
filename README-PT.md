# iPhone 17 Pro Max

Projeto feito com Vite + React.

## Como rodar

1. Instalar dependências:

```
npm install
```

2. Rodar em desenvolvimento:

```
npm run dev
```

3. Build (produção):

```
npm run build
```

4. Publicar no GitHub Pages (opção local):

```
npm run deploy
```

---

> Observação: este repositório também possui um workflow de GitHub Actions que faz deploy automático ao dar push na `main` (gera o build e publica o `dist/`).
